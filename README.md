# IAmCat-PoopSpawnerMod

I Am Cat にうんこをスポーンできるようにする非公式 MelonLoader MOD です。

## Features

- 右コントローラーのBボタンでうんこをスポーン
- 掴める
- 押せる
- NPCに食べさせられる

## Requirements

- I Am Cat PC版
- MelonLoader 0.7.3
- Windows

## Installation

1. I Am Cat に MelonLoader を導入する
2. Releaseページから `IAmCat-PoopSpawnerMod-v1.0.0.zip` をダウンロードする
3. zipを展開する
4. `Mods/IAmCatPoopSpawnerMod.dll` をゲームの `Mods` フォルダに入れる
5. ゲームを起動する

## Controls

- Right Controller B Button: Spawn Poop
